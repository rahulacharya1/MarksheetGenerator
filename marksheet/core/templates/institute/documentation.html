System Guide v1.0

Welcome to the EduMarks Documentation.

This guide will help you navigate the institutional portal, from setting up your first classroom to generating professional student transcripts.

01 Classroom Structure

Before adding students, you must define your school's structure. This allows the system to group results accurately.

Defining Sections

Navigate to Management > Class Structure. Here you can add class names (e.g., 10th) and sections (e.g., A, B, C). This is a one-time setup per academic year.

02 Student Enrollment

Once classes are ready, you can populate them with students. Each student is assigned a unique Roll Number for tracking.

Single Entry

Use the manual form to add individual students with their specific roll numbers.

Verification

Ensure the Student Name matches official records as it will appear on the final transcript.

03 Marks Management

The Marks Entry system is designed for speed. You can update scores subject by subject for an entire class.

Best Practices

  • • Always select the Class and Section first to load the correct student list.
  • • Marks are saved instantly upon submission of the update form.
  • • The system automatically flags scores below 35% as "Fail" on reports.

04 Professional Reports

EduMarks generates high-fidelity transcripts. These are "Print-Ready," meaning all navigation menus are hidden when you press Ctrl + P.

Tip: Use Google Chrome for the best print layout scaling.
